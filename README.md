## Getting Started

First, run the development server:

```bash

yarn dev
# or

# or
```

## Stripe Checkout (Demo)

1. Create a Stripe account and get your API keys.
2. Add the following environment variables to your deployment/dev environment:

```
STRIPE_SECRET_KEY=sk_live_or_test_xxx
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_or_test_xxx
```

3. Start the dev server and open `/stripe` to see the demo page.

Implementation details:
- API route: `app/api/stripe/create-payment-intent/route.ts`
- Component: `components/StripeCheckout.tsx` (vanilla Elements, inline style similar to Hotmart)
- Demo page: `app/(open)/stripe/page.tsx`
